# AUM Securities - Refer & Earn Module

This project is a **Refer & Earn** form module for AUM Securities, built using **React**, **Tailwind CSS**, and **Lucide React icons**. It allows users to submit their details to participate in the referral program. The module features a professional and responsive UI inspired by AUM Securities’ branding.

---

## Features

- **Responsive design**: Works seamlessly on desktop and mobile devices.
- **Equal-width layout**: Left panel shows referral benefits, right panel contains the form.
- **Professional UI**: Uses AUM Securities color theme `#5064a8` with clean typography.
- **Referral form**:
  - Full Name
  - Mobile Number
  - Email Address
  - Aadhaar Number
  - PAN Number
  - Referral Code (optional)
- **Icons**: Uses Lucide React icons for all input fields.
- **Referral benefits panel**: Lists professional, concise referral benefits for the user.
- **Form validation**: Required fields are enforced.
